
# Semgrep

A feature to instal Semgrep CLI in the devcontainer

## Example Usage

```json
"features": {
    "ghcr.io/sonikro/devcontainer-features/semgrep:1": {
        "version": "1.14.0"
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| version | Semgrep version to be installed | string | 1.14.0 |
